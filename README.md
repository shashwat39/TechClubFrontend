# Tech Club Frontend

This project is a responsive multipage website developed using HTML, CSS, and JavaScript. It aims to showcase the activities and offerings of the tech club of IIT Ropar. The website provides information about the club, its courses, frequently asked questions, and student testimonials. It also includes a responsive navigation bar, header section, categories section, popular courses section, FAQs section, and a section for student testimonials.

## Features

- Responsive design: The website is optimized for various screen sizes, ensuring a seamless experience across different devices.
- Navigation bar: The website includes a navigation bar that allows users to easily navigate between different pages.
- Categories section: The website showcases different categories of technology-related activities offered by the club, such as web development, app development, open source development, cloud computing, blockchain, and machine learning.
- Popular courses section: The website highlights popular courses offered by the club, providing brief descriptions and links for users to learn more.
- FAQs section: The website includes frequently asked questions about the tech club, covering topics such as the purpose of the club, membership, activities, prerequisites, and more.


## Technologies Used

- HTML
- CSS
- JavaScript
- ICONSCOUT CDN
- Google Fonts (Montserrat)
- Swiper JS

## Setup
To set up the project locally, follow these steps:
- Clone the repository: `git clone <repository-url>`
- Navigate to the project directory: `cd <project-directory>`
- Open the `index.html` file in a web browser.
